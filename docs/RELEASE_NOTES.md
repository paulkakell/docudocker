# Release notes

## v01.00.00

Initial release of DocuDocker Web App.

Highlights:

- Dynamic dark-theme dashboard for Docker documentation.
- Overview, Resources, Ports, and Mounts tabs.
- Sorting on every column and per-column filter dropdowns.
- Docker socket and docker-socket-proxy connectivity modes.
- Optional basic auth, structured logs, health endpoint, and security headers.
- Container build assets, compose examples, and validation documentation.
