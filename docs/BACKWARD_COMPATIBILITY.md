# Backward compatibility review

This is the initial `01.00.00` release. There is no prior external API contract to preserve.

The following interfaces should be treated as stable for future versions unless a breaking change is explicitly declared:

- `/api/overview` response fields
- `/api/resources` response fields
- `/api/ports` response fields
- `/api/mounts` response fields
- Environment variable names documented in `README.md`
- Footer GitHub link and version presentation
